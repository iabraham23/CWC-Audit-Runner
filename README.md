This repo manages the live website to run composite automations for CWC Advisors 

Input composite CSV's from Orion and download an excel sheet of a CWC style composite
Required CSV columns: 
- Included
- Reigstration Name
- Beginning Value
- Ending Value
- Gross Performance %
- Account Number
- Account Start Date
